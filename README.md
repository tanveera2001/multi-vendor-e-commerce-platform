## 🧩 Project Description

The **Multi-Vendor E-Commerce Platform** is a web-based application where multiple sellers can register and manage their own online stores within a single marketplace.

Customers can browse products, add them to their cart, place orders, and communicate with sellers. The platform also includes an admin dashboard to manage sellers, products, and system operations.
